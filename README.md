# Simple Tic Tac Toe Game

## Description
This project is a straightforward implementation of the classic **Tic Tac Toe** game using **Java Swing**.  
It was built as a practice exercise to strengthen skills in GUI programming, event-driven logic, and integrating backend game mechanics with a user interface.

The game provides:
- A clean 3x3 grid built with `JButton` components
- Turn-based play between two players (X and O)
- A start/reset button to restart the game
- Status updates showing whose turn it is
- Win and draw detection with pop-up messages (`JOptionPane`)
